- dup2 - allocates a new file descriptor that refers to the same open file description as the descriptor. It uses the descriptor number specified by the user.

- creat - create a new file or rewrite an existing one

- pipe - creates a pipe, a unidirectional data channel that can be used for interprocess communication

- fork - creates a new process by duplicating the calling process. The new process is referred to as the child process

- pthread_create - function starts a new thread in the calling process. 

- system -uses fork(2) to create a child process that executes the shell command

- printf - writes the output to the standard ouput stream

- The last line in invoke cmd checks if the command is in the command table and then checks the arguments
